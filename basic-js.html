Шпаргалка

      Типы данных + свойства типов, "исключения" + методы:

    • JavaScript имеет следующие примитивные типы данных:

      Числа (Number): целые числа и числа с плавающей точкой.

    • - Свойства и методы: Number.MAX_VALUE, Number.MIN_VALUE, toFixed(), toPrecision().

      - Строки (String): последовательности символов.

      - Свойства и методы: length, toUpperCase(), toLowerCase(), charAt().

      - Булевый (Boolean): true или false.

      - Свойства и методы: Boolean.prototype.toString(), Boolean.prototype.valueOf().

      - null и undefined: представляют отсутствие значения.

      - Символы (Symbol): уникальные и неизменяемые значения.

      - Методы: Symbol.for(), Symbol.keyFor().

      - Объекты (Object): контейнеры для свойств и методов.

      - Свойства и методы: Object.keys(), Object.values(), Object.hasOwnProperty().

    • Структуры данных:

      - Массивы (Array): упорядоченные списки элементов.

      - Методы: push(), pop(), shift(), unshift(), splice(), concat().

      - Очередь (Queue): структура данных, работающая по принципу "первый вошел - первый вышел".

      - Методы: enqueue(), dequeue(), peek().

    • Часто используемые функции библиотеки Math:

      - Math.random(): возвращает случайное число от 0 до 1.

      - Math.floor(): округляет число вниз до ближайшего целого.

      - Math.ceil(): округляет число вверх до ближайшего целого.

      - Math.max(): возвращает наибольшее число из набора чисел.

      - Math.min(): возвращает наименьшее число из набора чисел.

    • Арифметические операции:

      - Сложение: +.

      - Вычитание: -.

      - Умножение: *.

      - Деление: /.

      - Взятие остатка: %.

      - Инкремент: ++.

      - Декремент: --.

      Функции lodash:

      - Lodash - это библиотека JavaScript, предоставляющая удобные функции для работы с данными и коллекциями.

      - Некоторые функции lodash: _.map(), _.filter(), _.reduce(), _.sortBy(), _.debounce(), _.cloneDeep().

      Логические операции:

      - Логическое И: &&.

      - Логическое ИЛИ: ||.

      - Логическое НЕ: !

      Алгоритмические конструкции:

      - Условное выражение (if-else): позволяет выполнять код в зависимости от условия.
